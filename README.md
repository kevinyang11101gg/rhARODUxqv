# 项目简介

## 核心功能
- 用户协同过滤推荐算法
- 旅游数据爬虫采集
- 数据可视化分析

## 适用场景
- 旅游行业数据分析
- 旅游推荐服务
- 旅游市场研究

## 技术特点
- 使用Python语言进行开发
- 采用Django框架搭建后端
- 利用MySQL数据库存储数据
- 集成requests网络爬虫技术
- 应用机器学习算法进行推荐
- 使用BootStrap进行前端开发
- 通过ECharts实现数据可视化
